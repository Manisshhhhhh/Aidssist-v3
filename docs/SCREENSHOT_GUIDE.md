# Aidssist V3 Screenshot Guide

Use a clean browser window, preferably at 1440px wide for desktop captures and one narrow mobile capture for responsiveness. Avoid showing real secrets, local private data, or browser tabs with sensitive content.

## 1. Upload Page

- Capture: hero heading, upload card, dataset registry, API status.
- Suggested crop: full first viewport.
- Caption: "Aidssist v3 upload workspace for CSV and Excel datasets."

## 2. Dataset Registry

- Capture: recent dataset list with at least one uploaded sample dataset.
- Suggested crop: registry card and surrounding upload area.
- Caption: "Dataset registry keeps uploaded datasets available for analysis."

## 3. Dashboard Overview

- Capture: dashboard header and overview metric cards.
- Suggested crop: top dashboard section after analysis completes.
- Caption: "Analysis dashboard summarizes dataset size, insights, and recommendations."

## 4. Data Quality Panel

- Capture: quality score, missing cells, duplicates, empty columns, constant columns.
- Suggested crop: quality panel only, with enough surrounding margin.
- Caption: "Data quality checks flag missing values, duplicates, and structural issues."

## 5. Charts Panel

- Capture: rendered charts with chart title and recommendation context.
- Suggested crop: one or two chart cards.
- Caption: "Recommended charts are backed by real chart-data endpoints."

## 6. Forecast Panel

- Capture: controls, forecast chart, metrics, assumptions, warnings.
- Suggested crop: forecast section after a forecast is generated.
- Caption: "Forecasting shows model output with metrics, assumptions, and warnings."

## 7. Ask-Your-Data Chat

- Capture: chat prompt, answer, structured result table or metric.
- Suggested crop: ask-your-data panel with one or two example questions.
- Caption: "Deterministic dataset Q&A answers grounded questions without code execution."

## 8. AI Summary Panel

- Capture: AI summary options and generated result, or disabled-state message if LLM is off.
- Suggested crop: full AI summary panel.
- Caption: "Optional AI summaries explain deterministic Aidssist outputs."

## 9. Report Export Panel

- Capture: report options and successful download result.
- Suggested crop: report export card.
- Caption: "HTML and JSON reports package analysis outputs for sharing."

## 10. Workspace Switcher/Auth View

- Capture: workspace switcher in the header, or login/register screen when user auth is enabled.
- Suggested crop: header plus the workspace menu or auth page.
- Caption: "Workspace and authentication foundations support user-owned datasets."

## 11. GitHub Repo Screenshot

- Capture: GitHub repository home with README visible.
- Suggested crop: repo title, latest commit, README opening.
- Caption: "Aidssist v3 release candidate published on GitHub."

## 12. Docker Smoke/CI Screenshot

- Capture: terminal output from `make docker-smoke` or GitHub Actions workflow run.
- Suggested crop: PASS lines and final success state.
- Caption: "Docker smoke and CI checks verify the release-candidate build."
